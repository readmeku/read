# Project Hosting Instructions for Medical Center UOJ

## Database Information
- **Database Name**: medical_center
- **Database Server**: MariaDB 10.4.28
- **Default Character Set**: utf8mb4


## Hosting Instructions
1. Set up a local or online server environment with PHP version 8.2.4 or compatible.
2. Import the `medical_center_uoj.sql` file to create the required database structure.
3. Make sure to update any database credentials in the application configuration if they differ.
